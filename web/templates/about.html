Como usar

Todas as ligacoes no telefone fixo terminam com #. O # e a tecla de enviar.

Ligar para outra crianca

Para duas criancas se ligarem, as duas familias precisam autorizar na aba Contatos. Quando os dois lados autorizarem, o status fica verde.

Depois disso, basta discar o ramal da outra crianca. Por exemplo, se o ramal for 123, discar 123#.

Ligacao em grupo

Para abrir uma chamada e permitir que outros entrem, basta acrescentar um 0 antes do ramal.

Por exemplo, se normalmente voce discaria 123#, para abrir em grupo disca 0123#.

A chamada funciona normalmente entre os dois. Se uma terceira crianca ligar para qualquer um dos dois, ela entra na conversa automaticamente — desde que tenha autorizacao com todos que ja estao na chamada. Quem entra ouve um aviso dizendo quantas pessoas estao na chamada. A chamada aceita ate 5 participantes.

Sem o zero na frente, a chamada e privada e ninguem mais pode entrar.

Ligar para a familia

Voce pode configurar ate dois celulares na aba Ligacao para celular da familia. Cada celular recebe um ramal proprio. O ramal aparece no site apos a ativacao.

A crianca disca esse ramal no telefone fixo para ligar para o celular, e a familia pode ligar de volta pelo app.

Como funciona

O que e?

Um telefone fixo para as criancas ligarem umas para as outras. A crianca usa um telefone com fio comum, conectado a um pequeno aparelho (adaptador) que se liga a internet da casa. As ligacoes sao feitas pela internet, sem custo de operadora.

Controle das familias

Este site e o painel de controle. Aqui voce configura:

  • Horarios permitidos: quando o telefone pode ser usado
  • Contatos autorizados: quem pode ligar para quem
  • Celular da familia: para a crianca ligar para o celular de casa
Quanto custa?

O projeto e inteiramente gratuito. O servidor roda na camada gratuita da Oracle Cloud e o software e todo open-source (Asterisk, Flask, Linux).

O unico investimento e o aparelho adaptador (Grandstream HT801 ou HT802) e um telefone com fio.

Detalhes tecnicos

O adaptador Grandstream converte o telefone analogico em VoIP usando o protocolo SIP sobre TCP. Ele se conecta a um servidor Asterisk hospedado na Oracle Cloud (free tier). Este site roda Flask com banco SQLite no mesmo servidor. Cada chamada passa por um script AGI que consulta o banco para verificar horarios e permissoes em tempo real.

O projeto e open-source: github.com/celacanto/telefone-fixo
