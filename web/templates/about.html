Como usar

Todas as ligações no telefone fixo terminam com #. O # é a tecla de enviar.

Ligar para outra criança

Para duas crianças se ligarem, as duas famílias precisam autorizar na aba Contatos. Quando os dois lados autorizarem, o status fica verde.

Depois disso, basta discar o ramal da outra criança. Por exemplo, se o ramal for 123, discar 123#.

Ligação em grupo

Para abrir uma chamada e permitir que outros entrem, basta acrescentar um 0 antes do ramal.

Por exemplo, se normalmente você discaria 123#, para abrir em grupo disca 0123#.

A chamada funciona normalmente entre os dois. Se uma terceira criança ligar para qualquer um dos dois, ela entra na conversa automaticamente — desde que tenha autorização com todos que já estão na chamada. Quem entra ouve um aviso dizendo quantas pessoas estão na chamada. A chamada aceita até 5 participantes.

Sem o zero na frente, a chamada é privada e ninguém mais pode entrar.

Ligar para a família

Você pode configurar até dois celulares na aba Ligação para celular da família. Cada celular recebe um ramal próprio. O ramal aparece no site após a ativação.

A criança disca esse ramal no telefone fixo para ligar para o celular, e a família pode ligar de volta pelo app.

Como funciona

O que é?

Um telefone fixo para as crianças ligarem umas para as outras. A criança usa um telefone com fio comum, conectado a um pequeno aparelho (adaptador) que se liga à internet da casa. As ligações são feitas pela internet, sem custo de operadora.

Controle das famílias

Este site é o painel de controle. Aqui você configura:

  • Horários permitidos: quando o telefone pode ser usado
  • Contatos autorizados: quem pode ligar para quem
  • Celular da família: para a criança ligar para o celular de casa
Quanto custa?

O projeto é inteiramente gratuito. O servidor roda na camada gratuita da Oracle Cloud e o software é todo open-source (Asterisk, Flask, Linux).

O único investimento é o aparelho adaptador (Grandstream HT801 ou HT802) e um telefone com fio.

Detalhes técnicos

O adaptador Grandstream converte o telefone analógico em VoIP usando o protocolo SIP sobre TCP. Ele se conecta a um servidor Asterisk hospedado na Oracle Cloud (free tier). Este site roda Flask com banco SQLite no mesmo servidor. Cada chamada passa por um script AGI que consulta o banco para verificar horários e permissões em tempo real.

O projeto é open-source: github.com/celacanto/telefone-fixo
